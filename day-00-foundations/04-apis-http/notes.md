# APIs & HTTP
- **REST:** resources at URLs, acted on with verbs.
- **Verbs:** GET (read), POST (create/act), PUT/PATCH (update), DELETE.
- **JSON:** the body format every modern API speaks.
- **Status codes:** 2xx ok, 4xx your fault, 5xx server fault.

Run `python3 call_api.py`; read `mini_server.py` to see how to build one.
