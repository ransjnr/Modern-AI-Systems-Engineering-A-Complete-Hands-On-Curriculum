# ML fundamentals (conceptual)

- **Supervised:** learn from labelled examples (input → known output).
- **Unsupervised:** find structure without labels (e.g. clustering).
- **Train/test split:** hold out data to measure real performance.
- **Overfitting:** memorises training data, fails on new data — detect via a train/test score gap.
- **Metrics:** MAE/RMSE (regression); accuracy/precision/recall/F1 (classification).

Run `python3 tiny_model.py` for an end-to-end example.
