# Python for ML — the patterns the sprint uses daily

- **Scalars:** `int`, `float`, `str`, `bool`, `None`
- **Containers:** `list` (ordered, mutable), `dict` (key→value, = JSON), `tuple` (immutable), `set` (unique)
- **Comprehensions:** `[x*2 for x in data if x > 0]`
- **Functions & type hints:** `def predict(x: float) -> float:`
- **Classes:** model components, agents, schemas
- **Context managers:** `with open(...) as f:`
- **f-strings:** `f"MAE = {mae:.2f}"`

Run `python3 examples.py` to see every concept in action.
