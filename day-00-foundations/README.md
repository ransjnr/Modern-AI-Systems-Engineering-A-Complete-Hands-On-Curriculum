# Day 0 — Pre-Sprint Foundations

> Complete this **before Day 1**. Self-paced, ~10–14 hours. It exists because the sprint moves fast — roughly one production system per day. Arrive comfortable with Python, the terminal, Git, and ML concepts.

## Sections
| Folder | Topic | Time |
|--------|-------|------|
| `01-python/` | Python for ML: types, functions, classes, comprehensions, JSON, type hints | ~3 h |
| `02-terminal-git/` | Terminal navigation + Git & GitHub workflow | ~2 h |
| `03-ml-fundamentals/` | Supervised vs unsupervised, train/test, overfitting, metrics | ~2 h |
| `04-apis-http/` | REST, HTTP verbs, JSON, calling & building APIs | ~1 h |
| `exercises/` | Hands-on checks for every section | — |

## Readiness checklist (you should be able to…)
- [ ] Navigate directories with `cd`, `ls`, `pwd`
- [ ] Write a Python function with arguments and a return value
- [ ] Create and use a Python class
- [ ] Use a list/dict comprehension
- [ ] Parse JSON in Python
- [ ] Create a Git repo, commit, and push to GitHub
- [ ] Explain supervised vs unsupervised learning
- [ ] Explain overfitting and how to detect it
- [ ] Call a REST API with `requests`
- [ ] Have Python 3.11+, Git, and Docker installed (see ../SETUP.md)

## How to use
Read each section's `notes.md`, run the example scripts, then do the matching files in `exercises/`. Run any script with `python3 path/to/file.py`.
